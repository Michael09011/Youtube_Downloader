# YouTubeダウンローダー ユーザーガイド

## プログラム概要
このプログラムは、YouTubeからビデオまたはオーディオ（音楽）をダウンロードするためのGUIアプリケーションです。

## 主な機能
- ✓ YouTubeビデオのダウンロード（最高品質）
- ✓ オーディオのみをMP3としてダウンロード
- ✓ ダウンロード場所を自由に選択可能
- ✓ 直感的なGUIインターフェース
- ✓ リアルタイムのダウンロード進行状況表示

## 実行方法

### 方法1: EXEファイルで実行（推奨）
```
dist\YouTube_Downloader.exe
```
- Windows エクスプローラーで `dist\YouTube_Downloader.exe` ファイルをダブルクリックするだけです。
- 追加のインストールや設定は必要ありません。

### 方法2: Pythonで直接実行
```
python youtube_downloader.py
```

## 使用方法

1. **YouTube URLを入力**
   - ダウンロードしたいYouTubeビデオのURLを「YouTube URL」入力ボックスに貼り付けます。
   - 例: https://www.youtube.com/watch?v=dQw4w9WgXcQ

2. **保存場所を設定**
   - 「保存場所」入力ボックスに目的のフォルダパスを入力します。
   - 「参照」ボタンをクリックしてフォルダを選択することもできます。
   - デフォルト: `C:\Users\[ユーザー名]\Downloads\YouTube`

3. **ダウンロードオプションを選択**
   - **ビデオをダウンロード**: 元のビデオファイルをダウンロードします（推奨品質）
   - **オーディオのみをダウンロード（MP3）**: オーディオトラックのみをMP3形式でダウンロードします。

4. **ダウンロードを開始**
   - 「ダウンロード」ボタンをクリックします。
   - 進行状況は下部のログウィンドウに表示されます。
   - ダウンロード完了時に「ダウンロード完了！」メッセージが表示されます。

5. **パスをリセット**（オプション）
   - 「パスをリセット」ボタンをクリックすると、保存場所がデフォルト値にリセットされます。

## システム要件

### EXEファイル実行の場合
- Windows 7以降
- インターネット接続
- 最低100MBの空きディスク容量（実際の容量はダウンロードファイルによって異なります）

### Pythonで実行する場合
- Python 3.8以降
- 以下のパッケージをインストール必須：
  ```
  pip install yt-dlp PySimpleGUI ffmpeg-python
  ```

## トラブルシューティング

### 1. 「FFmpeg not found」エラー
- FFmpegがインストールされていません。
- Windowsコマンドプロンプトで以下を実行してください：
  ```
  pip install ffmpeg-python
  ```
  または https://ffmpeg.org/download.html から直接インストールしてください。

### 2. 「ネットワークエラー」
- インターネット接続を確認してください。
- YouTubeサーバーの状態を確認してください。
- URLが正しいか確認してください。

### 3. 「保存場所エラー」
- 入力したパスが存在するか確認してください。
- パスに特殊文字が含まれていないか確認してください。
- ディスクの空き容量を確認してください。

### 4. オーディオダウンロードがMP3でない場合
- FFmpegが正しくインストールされていない可能性があります。
- 上の「FFmpeg not found」の解決方法を参照してください。

## 重要な注意事項

1. **著作権**: YouTubeからダウンロードしたコンテンツの著作権は、元のクリエイターに属しています。
   個人使用以外の目的での許可なき配布は禁止されています。

2. **YouTubeの利用規約**: YouTubeのサービス利用規約を遵守する必要があります。

3. **ウイルス対策警告**: EXEファイルはアンチウイルスプログラムによってフラグが立てられる可能性があります。
   これはPyInstallerによる一般的な誤検知です。

## ライセンス

このプロジェクトは個人的な使用目的に限り、自由に使用できます。
